# EmployeeCRUD

CRUD de empleados en ASP.NET MVC.

##  Descripción
Este proyecto es un sistema web para gestionar empleados (crear, leer, actualizar y eliminar registros).  
Fue desarrollado como práctica siguiendo un tutorial, utilizando ASP.NET MVC y MySQL como base de datos.

##  Tecnologías utilizadas
- C#  
- ASP.NET MVC  
- MySQL  
- HTML  
- CSS  
- Bootstrap
